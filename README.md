## Feature

**사용자에게 거북목 관리 + 최적 환경 제공** 

1. 거북목 관리
   - 사용자 모방하여 꼬북목 제어
   - 위험 자세 일정 시간 유지 시, 경고 알림
   - 웹에서 자세 점수와 통계 확인
2. 시간 관리
   - 웹에서 타이머를 이용한 작업 및 휴식 시간 설정
   - 알림 (무음, 테마)
3. 습도 관리
   - 습도 정보 확인
   - 희망 습도 설정
   - 가습기 제어
